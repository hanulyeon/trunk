ML-Engine
*********

Building Engine
===============

Supported Platforms
-------------------

* Linux

You need to have a C++ compiler (supporting C++11) and ``make`` or ``gmake`` installed.

 $ sh /../../script/make-ml-engine.sh

Binaries
=========

* ML Master
$HD_HOME/modules/ml-engine/master/build/testMaster

* ML Worker
$HD_HOME/modules/ml-engine/worker/build/testWorker
